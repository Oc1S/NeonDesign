## API

| 属性      | 说明       | 类型          | 默认值 |
| --------- | ---------- | ------------- | ------ |
| className | 徽标类名 |        string       | - |
| style     | 徽标样式 | CSSProperties | - |
| count | 显示数字 | number | - |
| maxCount | 最大显示数字 | number | - |
| point | 鼠标是否为小手   | boolean | false |
| dot | 是否显示为小点   | boolean | false |
| showZero | count为0是否显示 | boolean | false |
